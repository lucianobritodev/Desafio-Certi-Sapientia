Desafio Certi Sapientia

Esta aplicação tem a finalidade de retornar números excritos por extenso. Para realizar este teste basta crescentar na url desta aplicação a barra seguida de um número qualquer, podendo ser positivo ou negativo conforme o exemplo a seguir:

Exemplos:

http://127.0.0.1:3000/-1042

ou

http://127.0.0.1:3000/94587
